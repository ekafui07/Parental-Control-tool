=====================================================
  PARENTAL CONTROL - HOW TO INSTALL
=====================================================

STEP 1: Install .NET 8 SDK (if not already installed)
  → https://dotnet.microsoft.com/download/dotnet/8.0
  → Download the SDK (not just Runtime)
  → Install it and restart your computer

STEP 2: Double-click INSTALL.bat
  → Click "Yes" when Windows asks for Admin permission
  → A PowerShell window will open
  → Wait ~1-2 minutes for it to build and install
  → You'll see green [OK] messages if successful

STEP 3: Check for errors
  → If the installer stops or shows red [FAILED] messages:
     - Open "install.log" in this folder
     - Read the error messages
     - Common issues:
       * .NET 8 SDK not installed (go back to Step 1)
       * Antivirus blocking the installer (disable temporarily)
       * Another parental control app running (uninstall it first)

STEP 4: Done!
  → A "Parental Control" shortcut appears on your Desktop
  → Blocking starts IMMEDIATELY — no setup needed
  → Service runs 24/7 in background

=====================================================
IMPORTANT: CHANGE THE ADMIN PASSWORD
=====================================================

  1. Open "Parental Control" from Desktop
  2. Go to "Settings" tab
  3. Change password from default: ParentAdmin123
  4. DO THIS BEFORE YOUR SIBLING SEES IT!

=====================================================
TROUBLESHOOTING
=====================================================

Problem: Installer stops immediately after admin prompt
  → Check install.log for errors
  → Most likely: .NET 8 SDK not installed

Problem: Sites still accessible after install
  → Open Desktop shortcut
  → Check if service status shows "Running"
  → If not running, try rebooting computer

Problem: Can't add/remove sites in manager
  → You need the admin password
  → Default is: ParentAdmin123

Problem: Want to temporarily disable blocking
  → Open Services (services.msc)
  → Find "Parental Control Service"
  → Stop it (requires admin password)
  → Start it again when needed
